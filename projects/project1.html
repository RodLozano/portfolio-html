Proyecto 1: Análisis de Datos Turísticos

Descripción: Este proyecto consiste en el análisis exploratorio de datos de ocupación hotelera en Castilla-La Mancha usando Python, pandas y visualización con matplotlib y seaborn.

Objetivo: Identificar patrones estacionales y predictores clave para modelos de demanda hotelera.

Tecnologías usadas: Python, pandas, Jupyter, seaborn, matplotlib, scikit-learn.
